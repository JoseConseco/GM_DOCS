Garments Library#

Garments Library gives you option to store and load your creations, with option to group cloth presets into categories (folder)

Defining library location#

To select library target folder location you have to go to: User Preferences -> Add ons -> Garment Tool -> Library tab Then select target folder location:

Managing Categories#

Triangle icon in Garments Library gives you option for managing library categories (folders):

  • Add new category
  • Delete selected category
  • Rename selected category

    Note - For Renaming and Deleting categories you have to select target categories by pressing '+' next to category name. You can rename, delete multiple folder by shift clicking on '+' icon

Managing preset#

To load garment preset into blend file just select garment preset icon, and press 'Append'. To Remove garment preset select garment preset icon, and press 'Remove'. To Rename garment preset select garment preset icon, and press 'Rename'.

Storing presets#

To store garment from current scene just select garment name, and press 'Save Garment' (located on bottom of panel). You can store multiple garments to one preset. For that just shift select garments names you want to save, and type in merged preset name.
